About

An open source no cost tool for creating customized SoC using No Code. This Mini SoC Generator will generate an System on Chip (SoC). It will be based on RISC-V ISA (IMFC extensions) (32-bit). It will take specifications of the SoC as parameters. All the Modules will be parameterized and reusable, for anyone who requires just a single part instead of an entire SoC. High Level approaches will be used by using CHISEL (Constructing Hardware in Scala Embedded Language)
